Google Pixel 7 国内使用指南

2024-03-10

上次去日本去体验了下 pixel 8，感觉还不错。问了之后才知道日本线下店几乎全部都是有锁机，只能使用日本电话卡，所以放弃。

我对 Google 手机的印象停留在很久之前用过的 Nexus 6，当时用作备用机，原生安卓系统确实给了我不一样的感受，不知时隔多年，现在又如何了。 这几天收到了新买的 Google Pixel 7，发现要折腾的东西还挺多，这里做一个记录，另外也许能帮助到手持 Pixel 7 的同学。

选择 Google Pixel 7 的理由

在目前在 Google 官网，日区已经下架了 Pixel 7，部分地区还有 Pixel 6a 但是已无库存。

从 Pixel 3a 开始，首次出现了 a 系列，更小的屏幕，更低的价格。但是在 Pixel 8 开始，只有 8 pro 和基础款，没有 8a。所以 Pixel 7a 也许是最后一代 Pixel a 系列手机。

Google 相机的算法似乎在内业都有很高的评价，不过我对于手机的拍照没有什么需求，能用就行。我也不怎么用手机拍风景照，毕竟专业的设备做专业的事，也不怎么玩手游。所以我主要是想选择一个不那么过时的 Google 手机，能体验到安卓原生系统是我最主要的原因。

Pixel 7 和 Pixel 7a 价格在淘宝浮动 100 左右，Pixel 7 有更大的屏幕（6.3英寸）以及更好的防水性能（IP 68）所以我选择了 Pixel 7。

Pixel 7 承诺有 3 年的软件更新，从发售至今持续到 2025 年 10 月。另外这次的新机 Pixel 8 承诺 7 年的软件更新，持续到 2030 年 10月。

新机验证

主要是从下面几个方面来验证:

  • 包装盒

  • IMEI查询

  • 气密检测

包装盒

主要是3码合一验证，包装盒 IMEI / 手机 IMEI / 系统中的 IMEI。取出 SIM 卡槽能看到 IMEI 。

IMEI 查询

这个是手机的唯一识别码，首先确定你买到的机器版本，因为 Google 官网 IMEI 查询需要区分版本，比如:

  • 日版 https://store.google.com/jp/repair

  • 美版 https://store.google.com/us/repair

查询到后检查下保修的时间是否正常。

注意需要登陆 Google 账号才能进行查询。

另外还有一个网站提供更详细的验证信息，不过我并不建议输入 IMEI 号到第三方的网站查询。 imei info: https://www.imei.info/phonedatabase/phones-google/

气密检测

我购买的 Pixel 7 是有 IP 68 级防尘防水，所以验证设备的气密信息可以有效的判断是否有过拆机。

主要是通过 Dev Check 软件检测压力传感器，查看数值是否正常。

打开软件后稍稍用力按压屏幕

正常来说没有拆封过的机器数值波动幅度会比较大。而拆机的设备，数据几乎不会有什么变化。

ROOT 刷入 Magisk

ROOT 不是必须，我是为了更好用。

相关软件 Magisk https://github.com/topjohnwu/Magisk

Android Platform Tool https://developer.android.com/tools/releases/platform-tools Android Driver （只有 windows 需要） https://developer.android.com/studio/run/win-usb

Pixel 7 全系列都是不锁 OEM 的，分为以下几个步骤:

  • 解锁 Bootloader

  • 下载线刷 ROM

  • 刷入线刷包

  • 安装 Magic 软件并打补丁

  • 刷入 init_boot

解锁 Bootloader

在 解锁之前，需要打开 OEM 锁，这个在开发者页面解锁就好，需要首次解锁需要梯子，否则按钮不可点击。

解锁 OEM 后，连接电脑，开启 USB 调试，输入命令：

# 重启手机 进入 bootloader 模式
adb reboot bootloader

# 进入 bootloader 模式之后
fastboot flashing unlock

执行完毕这些指令，就已经解锁了，在每次开机的时候会有一个提醒。告知你的手机已经解锁，不安全提示。

下载线刷 ROM

官方地址：https://developers.google.com/android/images 在这里选择你的型号，注意不要下载 OTA ROM。 下载后解压待用。

刷入线刷包

注意备份你的重要资料

这个时候把手机连接电脑，输入：

# 进入到解压的文件夹中

# 重启手机 进入 bootloader 模式
adb reboot bootloader

# mac 电脑 执行刷入
./flash-all.sh 

# windows 电脑
flash-all.bat

安装 Magic 软件并打补丁

Magic 软件下载：https://github.com/topjohnwu/Magisk

找到上面解压的线刷包里面的 init_boot 文件，把这个文件传输到手机上。然后使用 Magic 软件打补丁

按照以上的步骤，把打完补丁之后的文件传输到电脑上面。

刷入 init_boot

手机连接到电脑

# 重启进入 bootloader 模式
adb reboot bootloader

# 查看连接是否正常 （这里如果不能正常识别，一般在 Windows 电脑下，需要安装上面的驱动文件）
fastboot devices

# 刷入打好补丁的 init_boot 
fastboot flash init_boot 文件路径

等待刷入并自动重启，这个时候进入到 Magic 软件，可能会会提示需要对系统进行一些修补，点击确认后它会让手机再一次重启。之后就能正常看到当前 Magic 软件的版本，表示已经 root 成功。

开启电信 4G 支持

Pixel 7 支持一张实体卡和一张 esim，esim 在国内属于残废状态，Pixel 系列几乎都不支持电信卡。这里可以借助软件实现电信卡 4G 支持，另外使用 5G 必须要 ROOT。我这里用的 4G 已经足够了。

需要用到的软件 Shizuku https://github.com/RikkaApps/Shizuku Pixel IMS https://github.com/kyujin-cho/pixel-volte-patch

打开 Pixel IMS 会提示 Shizuku 的授权，然后点击第二个选项操作打开 VoLTE 和 VoWiFi 即可。

然后你的 Pixel 就支持电信 4G 了。

NFC 门禁支持（ROOT）

官方的应用是不支持门禁录入的，这里需要借助一个软件 NFC卡模拟专业版 打开软件后使用比较简单，先打开读卡模式

然后把你的门禁卡放在手机背部，等待读取，读取到之后会有一个弹窗提醒你添加。添加完毕之后，单机卡片会有一个弹窗，点击生成 Magic 模块。

然后导入到 Magic 中，重启手机即可。 推荐把这个小组件加入到下拉菜单中，方便快速切换。

通话录音（ROOT）

相关软件 https://github.com/jacopotediosi/GAppsMod

这个功能对于国产手机来说，普遍都有了，小米手机甚至可以录制微信飞书等语音通话。在 PIxel 上来说，如果你没有 ROOT ，那么几乎不能实现。 由于在国外对于隐私的监管和保护比较严格，通话录音是一个比较敏感的功能。不过 Google 的通话应用对于一部分国家也有开启，但是录制之前会提前告知对方此次通话会录音。

Google 通话应用里面的开关，是可以去修改的，Google 是保存在了数据库中，由 Google Play 统一去管理这些数据，所以 这里介绍的软件，去修改 Google 内置功能开关，达到内置的录音效果。

打开软件之后，开启第一项即可。（可以看到第二个选项已经不适用，这是替换提示音）然后反复的停止 Google Play 并清空数据 然后重启。直到看到电话中出现以下功能

所以就算录音也会提前告知对方，所以这个方案在早期，是替换了内置提醒音为静音，达到没有提醒音的效果。不过现在新版本已经不适用了，开启了自动通话录音会提示对方。

Safetey 安全检测（ROOT）

相关软件 https://github.com/kdrag0n/safetynet-fix

如果 ROOT 了之后，发现 Google Play 有部分软件无法下载，（提示你的手机不适用）比如 ChatGPT，这个问题可以刷入面具模块解决。刷入之后停止 Google Play 并清理全部数据重新打开即可。

国内应用的使用建议

对于国内的生态情况，有点一言难尽。我建议大家安装黑域，这个不需要 ROOT 也可以使用。简单来说防止这些国内软件互相进程唤醒，上传数据。 不过对于流氓到家的应用，黑域可能也没有什么办法，比如

禁止后台唤醒之后，软件反复的频繁的唤醒，虽然每一次都被黑域拦截 但是这样只会更耗电，黑域建议取消。

我后面卸载了应用宝。

最后

如果你是小白，我完全不建议你购买。Google Pixel 系列手机只适合发烧友和信仰粉，毕竟要花一些精力去调教，当然如果你比较喜欢折腾，想体验原生 Andoird 系统，厌倦了国产系统的特色，或许 Pixel 可以给你带来不一样的感受。
